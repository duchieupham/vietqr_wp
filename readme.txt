== Description ==
This is VietQR VN plugin. It used for connect to VietQR payment system

== Installation ==
1. Upload the `vietqr-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the 'VietQR VN' settings page to customize the plugin.

== Frequently Asked Questions ==
= Can I use this plugin on multiple websites? =
Yes, you can use this plugin on any number of websites.

= How can I customize the plugin settings? =
After activating the plugin, go to the 'VietQR VN' settings page to customize the plugin.
